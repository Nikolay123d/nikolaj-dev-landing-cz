# Nikolaj Dev Landing CZ

Czech version of the Nikolaj Dev landing page and demo template portfolio.

## About

This repository contains a static Czech sales website for website templates and landing pages. The site presents ready demo templates, service packages, pricing, contact options and a Forminit lead form.

## What The Site Sells

- Landing pages for local businesses and services.
- Website templates adapted to a client's photos, text, menu, services and contacts.
- Small business websites with pricing, gallery, map, contact buttons and a request form.
- Basic MVP / mini-platform work when a simple landing page is not enough.
- Project rescue, audits and bug fixing.

## Main Services

- START website from a ready demo template.
- PLUS website with more sections, language support and analytics basics.
- PRO website for a larger launch or growth plan.
- Template adaptation for restaurants, bakeries, renovation, tours, delivery, beauty and other categories.
- Bug fixing and project cleanup.

## Local Run

Open `index.html` directly in a browser, or run a simple local static server from this folder.

Example:

```powershell
python -m http.server 8776
```

Then open:

```text
http://127.0.0.1:8776/
```
